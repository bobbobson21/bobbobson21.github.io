Unnio for unity

How to install

First download the latest version.

Then find the unity project you want to add Unnio to.

Then extract the archive to that projects package folder and restart the project if you still have it open and you may also want to revalidate all after you open it just to be on the safe side.

What is Unnio

Unnio is a tool developed for Unity to help developers, programmers, and artists plan out tasks through Unnio's Kanban board assets. Unnio boards are also capable of automating a lot of the board's processes through stack/column actions, which can change properties of a node/card, and node conditions can be used to move nodes around based on tests against your asset and code database.

This tool was developed by Kyle Edwards.

Why you should use it

The main reason why it should be used is because of its automated control systems which can help speed up game development as you wont have to spend time manually adding colours to nodes or moving them around your self. but Unnio's other features, such as the issues system, can also of great aid for tracking progress and for understanding what needs to currently be done

Unnio boards also dont have to exsit within a unity project so you can also put them in a shared network folder for group work

Tutorialization

Unnio can be very difficult to get used to for the first time with all its many features and bells and whistles and because of that there is a lot of tutorial help with most of it coming from the tutorial Index, its like a book of power points in unity, which dose a lot of the explaining.

There is also a tutorial that you can follow along with incase the tutorial index is also something you are struggling with.

Unnio issues

Unnio issues is one of the many moving parts of Unnio and it is designed to track issues within your project. This can be also made to work automatically thought unit testing but you can also manually declare issues within code or with in the current scene, though issue components, should you wish to do so.

Unnio issues is also contained within a different package from the main one so if you don't want to you can just delete or if you only want it you can delete everything else.

Unnio templates

The template viewer allows you to view a range of free templates that you can use to get started with. All you have to do is open the template viewer, wait a bit and then select the template you want and then you can just added it to your project by clicking the "add to project" button and after that it becomes an editable board in your project.

I am responsible for development of sixteen of the templates in the template viewer and there is at least two templates for every type of job using unity that I could think of.
